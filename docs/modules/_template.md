## 安装

### 通过 Web UI 安装（推荐）

1. 打开 LexHub Web 控制台：`http://localhost:3000`
2. 进入「**模块市场**」
3. 找到 **[模块名称]** 卡片，点击「**安装**」
4. 等待安装完成，卡片状态变为「**运行中**」

### 通过命令行安装

```bash
# （若支持 CLI 安装，在此提供命令）
lh module install <module-name>
```

## 卸载

在 LexHub Web 控制台中，点击应用卡片上的「**卸载**」按钮，或：

```bash
lh module remove <module-name>
```

> **注意**：卸载前请备份重要数据，卸载操作将删除模块数据目录。

## 故障排查

| 症状 | 可能原因 | 解决方法 |
|------|----------|----------|
| 安装失败 | 网络问题 / 依赖缺失 | 检查网络连接，查看 `lh log` |
| 无法访问服务 | 端口被占用 | 检查端口占用，修改配置中的端口号 |
| 模块崩溃 | 依赖不兼容 | 查看 `lh log` 获取详细错误信息 |
